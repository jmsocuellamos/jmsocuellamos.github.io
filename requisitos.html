Requisitos

Para poder utilizar el código expuesto en estos materiales es necesario la instalación de diferentes librerías de R. Para instalar dichas librerías es necesario ejecutar el código siguiente:

pkgs <- c('tidyverse', 'stringr', 'forcats', 'lubridate', 'magrittr', 'broom', 
          'datasets', 'kableExtra', 'mosaic', 'latex2exp', 'pubh', 'moonBook',
          'sjlabelled', 'sjPlot', 'nasaweather', 'tidymodels', 'lmtest','alr3',
          'reshape2', 'olsrr','ggfortify','mgcv','modelr')
install.packages(setdiff(pkgs, installed.packages()[,"Package"]), 
                 dependencies = TRUE)

Para algún análisis especifíco se utilizará alguna librería acesoria que será cargada con el comando require().

Para cargar las librerías podemos utilizar estas dos versiones:
